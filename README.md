# Config

A central place to store and manage configuration files, commands, and profiles for the tools and environments I work with. This repository helps streamline setup, ensure consistency, and improve productivity across systems.

## Contents

- Tool configurations
- CLI commands & aliases
- Profiles and environment setups
- Miscellaneous helper configs

## Usage

Clone the repository and use the relevant configuration files as needed for your setup.  
Each folder/file is intended to be self-contained and easy to adapt.
